== Description ==

WordPress plugin that allows easy ACH bank transfer payments via Stripe + Plaid integration, 
or accepts credit card payments via Stripe's checkout.js 

== Installation ==

1. Clone or download this repo to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
